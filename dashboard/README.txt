## Project Overview

The Mutual Fund Analytics Platform is a comprehensive data analytics project developed to analyze mutual fund performance, investor behavior, SIP trends, portfolio concentration, and risk metrics.

The project includes:

* Exploratory Data Analysis (EDA)
* Performance Analytics
* Risk Analytics (VaR & CVaR)
* Investor Cohort Analysis
* SIP Continuity Analysis
* Fund Recommendation System
* Power BI Dashboard
* Advanced Financial Analytics
